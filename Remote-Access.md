## MQTT support  

**Luciferin** supports `MQTT` and can be controlled via a smartphone or via a PC remotely.

#### Turn ON/OFF the LED strip remotely, apply light effects  

```yaml
{"state": "ON"}
```
```yaml
{"state": "OFF"}
```
```yaml
{"state": "ON", "effect": "rainbow"}
```

Those are the supported effects:
- GlowWorm
- bpm
- candy cane
- confetti
- cyclon rainbow
- dots
- fire
- glitter
- juggle
- lightning
- noise
- police all
- police one
- rainbow
- solid rainbow
- rainbow with glitter
- ripple
- sinelon
- solid
- twinkle
